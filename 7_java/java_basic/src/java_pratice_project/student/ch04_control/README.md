# Java 1장 ~ 5장 실습문제

**자바 시작하기, 데이터와 변수, 연산, 조건문, 반복문, 배열, 참조변수** 실습문제용 프로젝트입니다.

## 구성

- `src/practice/ch01_start` : 자바 시작하기
- `src/practice/ch02_variables` : 데이터와 변수
- `src/practice/ch03_operators` : 연산
- `src/practice/ch04_control` : 조건문/반복문
- `src/practice/ch05_array_reference` : 배열/참조변수

## 사용 방법

1. 각 `.java` 파일을 열어 문제 설명을 읽습니다.
2. `TODO` 부분을 직접 작성합니다.
3. 컴파일 후 실행 결과가 기대 결과와 맞는지 확인합니다.
